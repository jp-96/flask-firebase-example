# Flask-Firebase-Example

Python 3.7

## Installation

Install the extension with pip:

```sh
$ pip install flask-login
$ pip install git+https://github.com/klokantech/flask-firebase
```

## Usage

Run app.py and open http://localhost:8080/

```sh
$ python app.py
```
